## ✨ Features

- 🎨 Animated typing effect using `pv`
- 🌈 Random colorful output every time
- 🖼️ Random Neofetch ASCII distro art
- 📱 Customizable Social Media Links
- ⚡ Fast & Smooth Fish Shell + Starship
- 🔧 Easy to customize name, tagline, speed, etc.

```

```
